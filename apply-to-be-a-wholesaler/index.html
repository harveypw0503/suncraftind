Get started

Navigation Overview

You can move between questions or screens in the dialog box using the 2 buttons in the lower right corner. 

In the same way, you can press the buttons below on each page of the dialog box that appears once the requested fields are filled out. 

Example: Once you press "Continue" as shown in the previous image of the welcome screen, the following Wholesaler / Company / Requestor Information screen will be displayed. 

Requestor Contact Information

You can see that each field has an example of what type of information to input, once this is done a "Continue" or “OK” button will appear at the bottom.

(You can also press enter)
This process will be repeated on each page of the window / form.

Form Responses

Continuing with the process, once you complete the fields of the previous tab and continue, you  will be presented with the following screens in which there are 2 types of responses, entered manually by the user and selected from a list of options. 

In this case the title of the example is "Address". The (*) indicates that your answer is mandatory. 

Each screen has a title and description to guide you as to what information to provide.

The description is below the title, this indicates how you can input your answer, in the case of  The address indicates that it is recommended to input the city, state and zip code.
(This would be a manually typed answer) 

Once selected, the "OK" or "Continue" button will appear to proceed with the other questions in  The dialog box. 

In case of selecting "options", simply click on the option you wish to select. 

Reseller Permit Requests

W9 Tax ID Number

This field is required and should be filled with a W9 Tax ID Number

Once your Tax ID is introduced, the "OK" or "Continue" button will appear to proceed with the other questions in  The dialog box. 

Credit References

This field requires 3 credit references but doesn't have to be lengthy, it only requires to meet the criteria while containing the necessary information.

The form indicates how to add each one of them, separated by line breaks. (using Shift + Enter)
